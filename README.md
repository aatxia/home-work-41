# React use() 

Проєкт показує використання хуку `use()` в React.
Deploy: https://home-work-41-pearl.vercel.app/ 

##  Реалізований функціонал:
- **`use()` Hook:** Використано для  Promise в `MessageComponent`.
- **`Suspense`:** Забезпечує відображення fallback-елемента, поки Promise виконується.
- **`ErrorBoundary`:** перехоплює помилки, якщо Promise повертає `reject`, і коректно відображає помилки.
